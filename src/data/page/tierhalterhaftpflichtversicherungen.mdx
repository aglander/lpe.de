# Tierhalterhaftpflichtversicherungen für Hund(e) und Pferd(e)

Für jeden Tierbesitzer ein Muss und nicht umsonst ist diese Absicherung in vielen Bundesländern inzwischen eine Pflichtversicherung!
In manchen Bundesländern ist sie sogar eine Pflichtversicherung.

Jeder Tierhalter kann auf Schadenersatz in Anspruch genommen werden, sobald sein Tier einen Dritten schädigt. Nach §833 BGB gilt das selbst dann, wenn ihn kein Verschulden trifft! 

Sachschäden können schnell die eigenen finanziellen Möglichkeiten übersteigen, aber insbesondere bei Personenschäden können extreme Kosten auf Sie zukommen.
